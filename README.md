#### Status
- Experimental

#### Usage

Use the [public version](https://metrico.github.io/clickhouse-mate/) _(no data goes through the server)_ or build your own:

```
git clone https://github.com/metrico/clickhouse-mate
cd ./click-house-play
npm install
npm start
```
